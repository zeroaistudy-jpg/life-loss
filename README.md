# 人生損失ラボ (Life Loss Lab) - Quiz Edition

「損失ラボシリーズ」の第一弾ツールが、さらに「刺さる」クイズ形式に進化しました。
ユーザーの日常的な20の生活習慣を問い直し、将来的な機会損失（時間・金額・ベンツ換算）を可視化します。

## 主な機能
- **虚無への20問**: はい／いいえ／どちらでもない の3択で答える診断形式。
- **独自投影ロジック**: 現在の習慣が80歳まで続いた場合の損失を、現在の年齢から逆算。
- **ベンツ換算演出**: 損失金額を高級車（ベンツ 1,200万円）の台数に換算して表示。
- **レスポンシブデザイン**: モバイル端末でも快適にボタン操作が可能。
- **キャラクター演出**: 砂時計の精が診断結果に合わせて毒舌（皮肉）混じりのメッセージを提示。

## 構成
- `index.html`: UI・構造（広告表示用スペース付き）
- `style.css`: デザイン（ラボ風ダークテーマ・アニメーション）
- `script.js`: クイズ・計算・キャラクター制御ロジック

## デプロイ（GitHub Pages）
1. GitHubに新規リポジトリを作成します。
2. このフォルダの中身を全てアップロードします。
3. リポジトリの `Settings` > `Pages` から、Sourceを `Deploy from a branch` にして保存します。
4. 数分後、発行されたURLにアクセス可能です。
