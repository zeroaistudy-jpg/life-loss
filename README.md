# 人生損失ラボ (Life Loss Lab)

「損失ラボシリーズ」の第一弾ツールです。
ユーザーの無駄な時間を可視化し、将来的な機会損失を突きつけることで行動を促します。

## 構成
- `index.html`: UI・構造
- `style.css`: デザイン（ラボ風・ダークテーマ）
- `script.js`: 計算ロジック・キャラクター制御

## デプロイ（GitHub Pages）
1. GitHubに新規リポジトリを作成します。
2. このフォルダの中身を全てアップロードします。
3. リポジトリの `Settings` > `Pages` から、Sourceを `Deploy from a branch` にして保存します。
4. 数分後、発行されたURLにアクセス可能です。

## シリーズ展開のヒント
`script.js` 内の `LAB_CONFIG` を書き換えるだけで、以下のツールに容易に変更可能です：
- **睡眠損失ラボ**: 無駄な夜更かし時間を入力。
- **スマホ損失ラボ**: スクリーンタイムを入力。
- **先延ばし損失ラボ**: やるべきことを後回しにしている時間を入力。
